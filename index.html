My recent works

The "Calculator" project is a web application that allows the user to perform basic arithmetic operations - addition, subtraction, multiplication, and division - on numbers. Additionally, there is a percentage calculation function.

The application has a simple and intuitive interface, consisting of input fields for numbers and operation buttons. And particularly a neomorphic design that gives a realistic effect to the application. To use the calculator, you need to enter two numbers in the input field and select the desired operation by clicking the corresponding button. After that, the application will display the calculation result on the screen.

The JS calculator uses the JavaScript language to perform arithmetic operations and process user input. HTML and CSS are needed to create the effects of button pressing, color, and output.

Code: https://github.com/WarriorOFlonelinesS/The-calculator

The "Restaurant Website" project is a task to create a web page that represents a restaurant and its menu. The project requires the use of HTML and CSS languages to create the interface, page styles, and responsiveness.

The main goal of the project is to create an attractive and user-friendly website where users can easily find information about the restaurant and its menu, as well as place orders and make reservations.

In the project, HTML was used to create the basic structure of the web page, including the header, main content, menu, and footer. CSS allowed for the styling of the pages, including colors, fonts, sizes, and element placement. Additionally, it was used for making the website responsive. In the mobile version, a burger menu was added using JavaScript.

Code: https://github.com/WarriorOFlonelinesS/Restaurant
